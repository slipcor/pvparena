# Stats Command

## Description

This command displays the top X players in a specific statistics type.

## Usage Examples

Command |  Definition
------------- | -------------
/pa stats LOSSES | shows the top 10 losers
/pa stats WINS 5 | shops the top 5 winners

Valid values are :

- WINS
- LOSSES
- KILLS
- DEATHS
- MAXDAMAGE
- MAXDAMAGETAKE
- DAMAGE
- DAMAGETAKE
- NULL
